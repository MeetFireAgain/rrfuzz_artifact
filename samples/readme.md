# Samples

This artifact includes four packaged sample tracks.

1. `toy_record_replay/`
- minimal RR record/replay smoke sample

2. `toy_fuzz/`
- minimal end-to-end fuzzing smoke sample

3. `lightftp/`
- runnable x86-64 sample
- includes source, trace, config, and launcher

4. `totolink_boa/`
- firmware-oriented sample
- includes raw firmware image, prepared `sim_root`, trace, dictionary, profile template, and launcher
